# Фотографии экспертов

## Требования к фотографиям

- **Размер:** 200x200px (квадрат)
- **Формат:** JPG или PNG
- **Стиль:** Профессиональные фото, деловой стиль
- **Обрезка:** Круглая (CSS автоматически обрежет)

## Имена файлов

1. `danilov.jpg` - Данилов Денис Юрьевич (Эксперт по бетонным производствам)
2. `sentyakov.jpg` - Сентяков Евгений Александрович (Эксперт по промышленному оборудованию)
3. `lihachev.jpg` - Лихачев Виктор Романович (Эксперт по запуску и оптимизации)
4. `kalashnikov.jpg` - Калашников Роман Валериевич (Эксперт по рецептурам и технологиям)

## Пример использования

После добавления фотографий, в `index.html` раскомментировать строки:

```html
<img src="images/experts/danilov.jpg" alt="Данилов Денис Юрьевич" class="expert-photo">
```

И удалить строку с placeholder:
```html
<!-- <div class="avatar-placeholder">ДА</div> -->
```
