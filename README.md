# Лендинг "Аудит действующего бетонного завода"

Прототип лендинга для услуги технического аудита бетонных заводов.

## Технологии

- HTML5
- CSS3
- JavaScript

## Структура

```
LendVik/
├── index.html          # Главная страница
├── css/
│   └── style.css      # Стили
├── js/
│   └── main.js        # JavaScript
└── images/            # Изображения
```

## Деплой на GitHub Pages

Сайт доступен по адресу: https://nobi-k.github.io/LendVik/

## Локальный запуск

```bash
python -m http.server 8000
```

Затем откройте http://localhost:8000
